# Quick Professional Website Enhancement Prompt

**Transform my portfolio website into a world-class professional site with advanced features.**

## Current Site
- Static HTML/CSS/JS portfolio on GitHub Pages
- Local chatbot, service worker, basic animations
- Dark theme, fast-loading
- URL: https://www.matomembowene.co.za

## Required Enhancements

### 1. Performance (Lighthouse 95+)
- WebP/AVIF images with lazy loading
- Code splitting and dynamic imports
- Enhanced service worker with stale-while-revalidate
- Resource hints (preconnect, dns-prefetch)
- Core Web Vitals optimization (LCP < 2.5s, CLS < 0.1)

### 2. Advanced UI/UX
- Smooth micro-interactions (GSAP-style animations)
- 3D tilt effects on project cards
- Scroll-triggered animations with Intersection Observer
- Dark/light mode toggle with smooth transitions
- Sticky nav with scroll progress indicator
- Interactive skill visualizations
- Loading skeletons (not spinners)
- Magnetic button effects

### 3. Enhanced Components
- Interactive project demos (embedded)
- Code snippets with syntax highlighting
- Before/after project comparisons
- Animated metrics/counters
- Lightbox for image galleries
- Advanced filtering with smooth transitions
- Search functionality (client-side)

### 4. Chatbot Upgrades
- Streaming responses (typewriter effect)
- Message history (localStorage)
- Suggested follow-up questions
- Voice input (Web Speech API)
- Better error handling with retry
- Context-aware responses

### 5. Accessibility (WCAG 2.1 AA)
- Full keyboard navigation
- Screen reader optimization
- ARIA labels throughout
- Reduced motion support
- Focus management
- High contrast mode

### 6. SEO & Discoverability
- JSON-LD structured data (Person, Organization)
- Enhanced Open Graph tags
- Optimized sitemap.xml
- Semantic HTML structure
- Rich snippets support

### 7. Security & Privacy
- Optimized CSP headers
- Security headers (X-Frame-Options, etc.)
- Privacy-first analytics (self-hosted pattern)
- GDPR considerations

### 8. Contact & Engagement
- Multi-step contact form with validation
- Success/error animations
- Email integration (Formspree pattern)
- Spam protection

### 9. Advanced Features
- Interactive resume with downloadable PDF
- Project filtering by multiple criteria
- Share project links
- Tag system for projects
- Testimonials section (if applicable)

## Technical Requirements
- Must work on GitHub Pages (static only)
- No backend dependencies
- Maintain fast load times
- Progressive enhancement (works without JS)
- Privacy-first (no third-party tracking)
- Mobile-first responsive design

## Success Criteria
- Lighthouse: 95+ Performance, 100 Accessibility, 100 Best Practices, 95+ SEO
- Core Web Vitals: All green
- WCAG 2.1 AA compliant
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)
- Mobile optimized

## Implementation Approach
1. Start with performance and accessibility (foundation)
2. Add UI/UX enhancements (polish)
3. Implement advanced features (differentiation)
4. Test and optimize (quality assurance)

**Maintain existing design aesthetic while elevating it to professional standards. Prioritize user experience and performance over flashy features.**
