# Site images

Add these image files here so they appear on the live site (GitHub Pages):

1. **header-section-foreground.png** – hero/header section foreground
2. **proof-references-experience-foreground.png** – Proof & references, Recent Experience, and other section foregrounds (same image used in multiple sections)

You can download them from your [Google Drive folder](https://drive.google.com/drive/folders/1zds7NwTioQecX7nd3BxYGsOBLQZBjLwT?usp=drive_link) and place the files in this folder.  
If the file from Drive is named `Proof & references-Recent Experience-foreground.png`, rename it to `proof-references-experience-foreground.png` (no spaces or `&`) so the URL works everywhere.

Then run:
```bash
git add images/
git commit -m "Add foreground images for live site"
git push origin main
```
